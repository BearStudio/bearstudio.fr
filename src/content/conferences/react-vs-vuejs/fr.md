Aujourd’hui pour lancer un projet d’application web, qui n’est pas juste un petit prototype, on peut être amené à devoir choisir entre React ou VueJS.

Venez partager votre avis dans une bonne ambiance, l’idée étant de voir le bon et le moins bon dans les 2 approches. Car, spoiler alerte, il n’y a pas de silver bullet. Mais en partageant ensemble les forces et les faiblesses peut être que cela vous permettra de pouvoir mieux faire votre choix.

Prenons le temps de discuter un peu de ces 2 différents librairies JS tous ensemble.
