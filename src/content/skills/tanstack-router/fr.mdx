Chez BearStudio, nous avons longtemps utilisé React Router — une excellente solution qui nous a accompagnés sur de nombreux projets. Mais dans notre quête permanente de fiabilité, il manquait une brique : la possibilité de vérifier automatiquement que tous les liens et toutes les adresses de l'application sont correctement reliés entre eux.

C'est exactement ce que TanStack Router nous a apporté. Un lien qui pointe vers une page qui n'existe pas, une information manquante dans une adresse — on le sait avant même de lancer l'application. Dans la continuité de notre approche TypeScript, c'est une garantie supplémentaire que la navigation fonctionne comme prévu, partout.

Aujourd'hui, nous utilisons TanStack Router au sein de TanStack Start sur nos projets, notamment à travers Start UI Web, notre starter open source qui intègre cette brique par défaut. Ce qui compte pour nous, c'est cette couche de fiabilité en plus sur une partie de l'application qu'on a longtemps dû vérifier à l'oeil.
