Start UI is our open-source starter. A ready-to-use foundation for kicking off a web or mobile project without spending time on technical setup. Authentication, user management, internationalization, dark mode, advanced components — everything is there from day one.

We built it from years of experimentation on real-world projects. On every new project, we kept running into the same needs, going through the same setup steps. Start UI is our answer to that: capitalizing on proven technical choices so we never start from scratch. The starter evolves continuously, shaped by every project we work on.

For our clients, the benefit is tangible. Less time and budget spent on technical plumbing with no visible business value. Consistent quality from one project to the next. And a team that masters its stack a little better with every delivery. Start UI is available in web, native, and Figma editions — and a few external contributors help make it grow as well.
