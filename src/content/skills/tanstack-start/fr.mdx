Au BearStudio, TanStack Start est devenu notre meta-framework de référence pour les projets web. Il a remplacé Next.js dans notre starter open source Start UI Web, et c'est aujourd'hui la base sur laquelle nous démarrons tous nos nouveaux projets.

Ce qui nous a convaincus, c'est une approche qui colle à notre philosophie. Là où d'autres solutions misent sur l'automatisation, TanStack Start nous laisse comprendre et maîtriser ce qui se passe sous le capot. Moins de magie, plus de contrôle. Le tout repose sur des standards du web que l'on connaît bien, avec des technologies dans lesquelles nous avons déjà une solide expertise.

En pratique, TanStack Start nous permet de réunir le front et le back dans un seul projet, avec les mêmes technologies de bout en bout. Le partage d'informations entre les deux est naturel, le déploiement est simplifié, et l'ensemble forme un tout cohérent. C'est exactement le socle dont nous avions besoin pour livrer des applications web robustes et maintenables.
