At BearStudio, TanStack Start has become our go-to meta-framework for web projects. It replaced Next.js in our open-source starter Start UI Web, and is now the foundation on which we kick off every new project.

What won us over is an approach that aligns with our philosophy. Where other solutions lean heavily on automation, TanStack Start lets us understand and control what's happening under the hood. Less magic, more control. It's all built on web standards we know well, using technologies we already have deep expertise in.

In practice, TanStack Start lets us bring front-end and back-end together in a single project, with the same technologies end to end. Sharing information between the two is seamless, deployment is simpler, and everything forms a coherent whole. It's exactly the foundation we needed to ship robust, maintainable web applications.
