Ficus UI is our open-source component library for React Native. We built it because when we needed it, no existing solution offered an approach as simple and composable as what we were used to on the web with Chakra UI. Rather than compromise, we built our own.

The idea is straightforward: style a component by passing it properties, without writing lengthy stylesheets. Colors, spacing, borders, responsive behavior — everything is handled at the component level, in a readable and fast way. A theming system lets us adapt the entire interface to each project in just a few lines.

Ficus UI ships by default in Start UI Native, our mobile starter. It lets us prototype quickly, maintain visual consistency across the whole application, and have ready-to-use components for the most common needs. It's a tool born from our real-world needs, and one we keep improving with every project.
